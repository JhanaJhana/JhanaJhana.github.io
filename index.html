Ｈexo & Github

Posted on 2023-09-08 Edited on 2023-09-15 Disqus:

Hexo

使用環境：Node.js
優點：
1. 可使用Markdown
2. 豐富的主題 (此網站使用主題NexT)
3. 官方有中文說明文件
4. 強大的外掛系統

  • Node.js使Javascript從前端延伸到後端，讓電腦可以用 Command line 的方式執行 JavaScript

Download: Node.js & Git

電腦需先安裝Node.js 和 Git

Download: Hexo

在終端機上下載hexo

  • npm（node package manager）為node.js的套件管理工具

Run server

從本機端啟動伺服器

Deploy on Github

  • github：可以存放靜態網頁，用.github.io的子網域
    創建Github帳戶, 並創建新的Repository(儲存庫)
    回到_config.yml檔案中的Deployment，更改相對應Repository的網址 更改後再回到終端機上，把內容deploy到Github上面 –>More info: Deployment

改主題

假設改成一個主題叫NexT

再到_config.yml中的theme更改成next

  • 選擇NexT: 很有名、簡潔清楚、有在維護、有很多可以改的地方

引入外掛系統

選擇Disqus讓每一篇文章下方都可以有留言功能
去Disque的官網創帳號，並命名shortname
回到_config.yml檔案中，找到Disqus，把enable改成true

此時，每一篇文章下方都會有留言功能。
但是！Menu中的頁面（例如：About）也都會出現留言功能！
因此，需找到其他不需要留言功能的頁面index.md檔案中的Front-matter，修改comment設定。
