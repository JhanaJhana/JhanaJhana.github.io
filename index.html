如何用 Hexo 和 Github 搭建個人網誌

Posted on 2023-09-01 Edited on 2024-03-22 In Website Disqus:

放暑假看了很多分享NLP的blog之後想想，咦！要不我也來搭建一個人網誌吧！！

  • 前言： 雖然已經有 HackMD，但在 HackMD 上面主要寫自己上線上課程還有線下在校修課的Learning Notes。因此決定要一個自己的網站，預計可以作為分享閱讀心得或小專案分享使用。

第一篇就決定是：如何用Hexo和Github搭建網站！

Q: 為什麼選Hexo呢？

核心原因是因為想要一個乾淨簡潔的網誌，在看了網路許多分享，發現我喜歡的風格都是使用 Hexo 搭建的！
所以進一步了解，又發現了以下的優點：

  1. 可以使用 Markdown 語法！程式碼可以很簡潔的顯示。
  2. 方便的外掛系統。
  3. 有豐富簡潔、內容清晰排版的主題可以選擇。
  4. 可以放在 Github 的儲存庫。
  5. 而且免費。
  6. 想要嘗試看看大名鼎鼎的 Node.js 。

所以首先可以先去了解了一下 Node.js

  • Node.js 使 Javascript 從前端延伸到後端，讓電腦可以用 Command line 的方式執行 JavaScript
